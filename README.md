TechFUNのC言語勉強会資料
====================


####ディレクトリ構造
* /bin
  * ツール群
* /docs
  * 講座の資料のディレクトリ
* /docs/n/
  * n回目の資料を配置するディレクトリ
* /docs/template/
  * スライドのテンプレートとエンジンが入っている

####作成の心得
1. ライセンス違反はしない
2. 外部に公開できない情報はコミットしない
3. 嘘は書かない
4. 文章は、である調を使用し、体言止めはなるべく使わない
5. 誤字・脱字、プログラムエラーがある場合はIssueまたはPullRequestを送る
6. PullRequestはマージする前に必ず第3者のレビューを1回は通す
